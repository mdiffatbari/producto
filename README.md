App Name: Producto

Description:

Producto is a modern, responsive web application that allows users to explore, search, and install apps seamlessly. The platform showcases top apps, provides detailed app information, and visualizes app reviews with interactive charts. Users can manage their installed apps using a dedicated "My Installation" page, including installing, uninstalling, and sorting apps by popularity. The application is designed for an optimal experience on all devices, ensuring fast performance, error-free navigation, and a smooth, interactive user interface.

Technologies Used:

Frontend: React, React Router, TailwindCSS, DaisyUI
Data Visualization: Recharts
UI Components & Icons: React Icons
Notifications: React Toastify
Build Tool: Vite
